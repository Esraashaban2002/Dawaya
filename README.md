# 🩺 DAWAYA | داوايا

Dawaya is a healthcare platform that connects patients and pharmacies through a modern digital experience. The platform helps users search for medicines, manage profiles, upload prescriptions, and access pharmacy services efficiently.

---

## Features

### User Features

* User Registration
* User Login
* OTP Verification
* Password Recovery
* User Profile Management
* Medicine Search
* Prescription Upload
* WhatsApp Reminder Interface

### Pharmacy Features

* Pharmacy Dashboard
* Inventory Management
* Stock Management
* Order Management

### Admin Features

* Admin Dashboard
* User Management
* Pharmacy Management
* Platform Monitoring

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to project folder

```bash
cd dawaya-frontend
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=your_backend_url
```
